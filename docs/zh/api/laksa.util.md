# Laska.util

Laksa 的工具库，提供各种通用功能

#### 用法

```Javascript
const Laksa=require('laksa')
// 或者
import Laksa from 'laksa'

// 初始化，不过provider是否设置不影响util的使用
const laksa=new Laksa()

laksa.util.<{Method|Object|Class}>
```
